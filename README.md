# Camonkii
Pet Shop Online
 
## Descripcion
 
Camonkii fue desarrollado originalmente como proyecto final para un seminario tecnologico en 2025. En 2026 fue retomado y mejorado, incorporando mejoras en el estilo visual y en la estructuracion de componentes.
 
La aplicacion permite a los usuarios explorar y comprar productos para mascotas. Para acceder a todas las funcionalidades, el usuario puede registrarse mediante un formulario con validaciones.
 
## Funcionalidades
 
- Home con productos destacados
- Catalogo completo de productos con detalle individual
- Carrito de compras con valor acumulado en tiempo real
- Registro de usuario con validaciones
## Detalle de funcionalidades
 
**Home**
Pagina principal que presenta los productos mas vendidos de la tienda.
 
**Productos**
Lista la totalidad de productos disponibles. Cada producto cuenta con un boton para acceder a su informacion detallada: descripcion, precio e imagen.
 
**Carrito**
Panel donde el usuario visualiza todos los productos agregados y el valor total acumulado de la compra.
 
**Registro**
Formulario reactivo que permite crear una cuenta. Incluye validacion de nombre de usuario disponible contra el backend, validacion de formato de email, longitud minima de contrasenia y confirmacion de contrasenia con validacion cruzada.
 
## Tecnologias
 
- Angular 19
- TypeScript
- Angular Reactive Forms
- SCSS
- REST API
